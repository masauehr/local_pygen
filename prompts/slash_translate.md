# スラッシュコマンド: /translate — 言語変換

> 別のプログラミング言語に変換するコマンド。

## プロンプト本文

```
以下のコードを {target_language} に変換してください。
元のロジック・動作を忠実に再現し、変換後の言語らしい書き方（イディオム）を使ってください。
変換のポイントがあれば日本語で補足してください。

【変換元コード】
```
{{{ input }}}
```
```

## config.json への追加方法

```json
{
  "name": "translate",
  "description": "コードを別の言語に変換する",
  "prompt": "以下のコードを指定した言語に変換してください。元のロジックを忠実に再現し、変換先言語らしいイディオムを使ってください。変換のポイントを日本語で補足してください。\n\n変換先言語: （ここに言語名を書く）\n\n【変換元コード】\n```\n{{{ input }}}\n```"
}
```
