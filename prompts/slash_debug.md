# スラッシュコマンド: /debug — デバッグ支援

> Continue の `customCommands[].prompt` 用テンプレート。

## プロンプト本文

```
以下のコードとエラーメッセージを分析して、原因と修正方法を教えてください。

【コード】
```
{{{ input }}}
```

【エラーメッセージ（あれば貼り付けてください）】
（ここにエラーを貼る）

## 回答形式
1. **原因**: エラーの根本原因を1〜2文で
2. **修正コード**: 修正済みのコードを提示
3. **解説**: 修正箇所と理由を箇条書きで
```

## 使い方

1. VS Code でエラーのあるコードを選択
2. Continue チャットで `/debug` と入力
3. エラーメッセージをプロンプトに追記して送信

## config.json への追加方法

```json
{
  "name": "debug",
  "description": "エラー原因を特定して修正する",
  "prompt": "以下のコードとエラーメッセージを分析して、原因と修正方法を教えてください。\n\n【コード】\n```\n{{{ input }}}\n```\n\n## 回答形式\n1. **原因**: エラーの根本原因を1〜2文で\n2. **修正コード**: 修正済みのコードを提示\n3. **解説**: 修正箇所と理由を箇条書きで"
}
```
